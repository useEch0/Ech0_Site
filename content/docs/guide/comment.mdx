title: 评论接入
description: Ech0 评论系统说明，支持多种评论服务
---

Ech0 支持多种评论服务接入，同时也内建了评论系统，无需第三方服务即可使用。

## 支持的评论服务

- **Twikoo** — 轻量、快捷、无侵入
- **Waline** — 安全、快速的评论系统
- **Artalk** — 自托管评论系统
- **Giscus** — 基于 GitHub Discussions 的评论系统
- **内建评论系统** — 无需任何第三方服务

## 启用

在 Ech0 设置中进入`评论设置`，选择你想使用的评论服务并填写对应参数即可。不同服务的必填字段会在界面中动态展示。

如果你不需要第三方评论服务，也可以直接使用内建评论系统，只需在设置中启用评论开关即可。

## 管理

你可以在后台直接管理评论内容，包括审核、删除与状态调整。
