## 概述

Ech0 内建通知收件箱，自动收集系统事件和用户互动消息。

## 消息来源

| 来源 | 标识 | 说明 |
| --- | --- | --- |
| 系统通知 | `system` | 版本更新、备份完成等系统事件 |
| 用户通知 | `user` | 互动相关通知 |
| Agent 通知 | `agent` | AI 摘要生成完成 |

## 功能

- 已读/未读状态和未读计数
- 阅读次数追踪（`ReadCount`）
- 单条删除和全部清空
- 关键词搜索
- 分页查询

## 编辑器集成

在编辑器的模式面板中可直接切换到收件箱模式，无需离开编辑页面即可查看消息。
