## 概述

Ech0 的访问令牌用于 API 请求认证，通过 **系统设置 → 访问令牌** 生成和管理。

## 有效期选项

| 选项 | 标识 | 说明 |
| --- | --- | --- |
| 8 小时 | `8_hours` | 短期临时令牌 |
| 1 个月 | `1_month` | 中期令牌 |
| 永不过期 | `never` | 长期令牌 |

## 使用方法

在请求头中加入 `Authorization` 字段，使用 `Bearer` 方式：

```http
POST /api/echo HTTP/1.1
Host: your-ech0-instance.com
Authorization: Bearer eyJh****Bt-E
```

## 管理

- 每个令牌有唯一 Token 值和自定义名称
- 支持随时吊销（删除）
- 令牌列表展示创建时间和过期时间

<Callout type="warn">
令牌创建后仅展示一次完整值，请立即复制保存。令牌使用时请妥善保管，避免泄露。
</Callout>
