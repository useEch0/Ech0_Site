## 访问令牌概览

Ech0 的访问令牌用于 API 请求认证，**只能通过 Web 界面生成**。  
生成后，你可以在请求头中使用 `Bearer` 方式携带令牌进行调用。

### 示例令牌

| Token       | 名称       | 创建时间               | 过期时间  | 操作 |
| ----------- | ---------- | -------------------- | -------- | ---- |
| eyJh****Bt-E | 我的令牌    | 2025/10/19 16:20:25 | 永不过期 | -    |

### 使用方法

在请求头中加入 `Authorization` 字段，并使用 `Bearer` 方式：


```http
POST /api/echo HTTP/1.1
Host: your-ech0-instance.com
Authorization: Bearer eyJh****Bt-E
```

### 注意事项

- 访问令牌能通过 WebUI 创建。
- 令牌使用时请妥善保管，避免泄露。
- 令牌永不过期或根据创建时设定的过期时间生效。
