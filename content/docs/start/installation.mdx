# 🐳 Docker 部署（推荐）

```shell
docker run -d \
  --name ech0 \
  -p 6277:6277 \
  -v /opt/ech0/data:/app/data \
  -e JWT_SECRET="Hello Echos" \
  sn0wl1n/ech0:latest
```

> 💡 部署完成后访问 `http://<服务器IP>:6277` 即可开始使用。  
> 🚷 建议将 `-e JWT_SECRET="Hello Echos"` 中的默认值替换为高强度随机字符串。  
> 📍 首次注册账号会自动成为管理员（当前仅管理员可发布内容）。  
> 🎈 数据默认持久化在 `/opt/ech0/data`。

# 🐋 Docker Compose 部署

创建一个新目录，并将 `docker-compose.yml` 放入该目录。

在该目录下执行以下命令启动服务：

```shell
docker-compose up -d
```
