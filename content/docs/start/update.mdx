# 🔄 Docker 部署更新



<Steps>
<Step>

停止当前容器

```shell
docker stop ech0
```

</Step>


<Step>
移除容器

```shell
docker rm ech0
```
</Step>

<Step>
拉取最新镜像

```shell
docker pull sn0wl1n/ech0:latest
```

</Step>

<Step>
启动新版本容器

```shell
docker run -d \
  --name ech0 \
  -p 6277:6277 \
  -v /opt/ech0/data:/app/data \
  -e JWT_SECRET="Hello Echos" \
  sn0wl1n/ech0:latest
```
</Step>
</Steps>

# 💎 Docker Compose 部署更新

<Steps>
<Step>

进入 `docker-compose.yml` 所在目录
```shell
cd /path/to/compose
```

</Step>
<Step>
拉取最新镜像并重建服务
```shell
docker-compose pull && \
docker-compose up -d --force-recreate
```
</Step>
<Step>

清理旧镜像（可选）
```shell
docker image prune -f
```
</Step>

</Steps>
