# 🔄 Docker部署如何更新



<Steps>
<Step>

停止当前的容器

```shell
docker stop ech0
```

</Step>


<Step>
移除容器

```shell
docker rm ech0
```
</Step>

<Step>
拉取最新的镜像

```shell
docker pull sn0wl1n/ech0:latest
```

</Step>

<Step>
启动新版本的容器

```shell
docker run -d \
  --name ech0 \
  -p 6277:6277 \
  -v /opt/ech0/data:/app/data \
  -v /opt/ech0/backup:/app/backup \
  -e JWT_SECRET="Hello Echos" \
  sn0wl1n/ech0:latest
```
</Step>
</Steps>

# 💎 Docker Compose部署如何更新

<Steps>
<Step>

进入 compose 文件目录
```shell
cd /path/to/compose
```

</Step>
<Step>
拉取最新镜像并重启
```shell
docker-compose pull && \
docker-compose up -d --force-recreate
```
</Step>
<Step>

清理旧镜像 
```shell
docker image prune -f
```
</Step>

</Steps>
